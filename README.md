# vue-online-shop
## 技术：Vue、Vue CLI、Vue Router、Vuex、Axios.
## 功能描述： 
    + 浏览商品
    + 选择商品
    + 商品价格排序
    + 登录
    + 注册
    + 上传商品
    + 修改商品
    + 删除商品。
## 不足：
    1. 因为只是一个初级的版本，所以部分功能还没有开发完善。比如：收藏商品和购买过的商品。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
